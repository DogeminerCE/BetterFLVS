# BetterFLVS

BetterFLVS is a powerful userscript designed to overhaul the Florida Virtual School (FLVS) user interface, specifically optimized for mobile devices. It transforms the often-cluttered desktop layout into a sleek, touch-friendly mobile experience.

## 🚀 Features

-   **📱 Mobile-First Navigation:** A dedicated bottom navigation bar for quick access to Lessons, Assessments, Gradebook, and Email.
-   **🌙 Optimized Dark Mode:** A comprehensive dark theme with high contrast, designed to be easy on the eyes without "ghosting" images or media.
-   **⚡ Quick-Action FAB:** A Floating Action Button that provides instant access to your Profile, Announcements, and Dark Mode toggle.
-   **📖 Reading Experience:** Enhanced typography and centered layouts for course readings, making it easier to study on the go.
-   **💾 Auto-Save Drafts:** Never lose your work again. The script automatically saves your assessment answers to local storage and provides a "Restore" button if needed.
-   **🧹 Distraction-Free Mode:** Automatically hides bulky headers, sidebars, and unnecessary UI elements on mobile.
-   **🔗 Smart Navigation:** Custom buttons automatically map to the original FLVS navigation elements, ensuring compatibility even with session tokens.

## 🛠️ Installation

1.  **Install a Userscript Manager:**
    -   [Tampermonkey](https://www.tampermonkey.net/) (Recommended)
    -   [Greasemonkey](https://www.greasespot.net/)
2.  **Add the Script:**
    -   Click on the `betterflvs.user.js` file in this repository.
    -   Click the **Raw** button.
    -   Your userscript manager should prompt you to install the script.
3.  **Go to FLVS:**
    -   Log in to your FLVS account (VSA or Canvas).
    -   BetterFLVS will automatically activate on mobile-sized screens.

## ⚙️ Configuration

You can toggle features directly within the script or via the Floating Action Button menu on the bottom right of your screen.
